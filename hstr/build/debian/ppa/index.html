This is Personal Package Archive (PPA) for MindForger and HSTR.

To install programs from this PPA add new repository to /etc/apt/sources.list as follows:

      deb http://www.mindforger.com/debian stretch main

... and import key that is used to sign the release:

      $ wget -qO - http://www.mindforger.com/gpgpubkey.txt | sudo apt-key add -

After that you can install MindForger as any other software package:

      $ apt-get update
      $ apt-get install mindforger

Similarly you can install HSTR:

      $ apt-get update
      $ apt-get install hstr

Don't worry about squeeze part in repo name: package should work on Debian squeeze+, Ubuntu 10.0+. Package contains binary, man page and bash completion.
